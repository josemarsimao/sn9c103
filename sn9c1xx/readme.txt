Link para download do driver:
http://www.linux-projects.org/download/sn9c10x-driver/


1 - O arquivo 'sn9c1xx-1.50.tar.gz' contém o driver original baixado do link

2 - A pasta 'modified' contém a verão adaptada com algumas alteraçãoes para que fosse compilado.
    Porém precisa de ajustes para funcionar.

3 - A pasta 'aio' contém uma versão consolidada em um único arquivo com o mesmo código da pasta 'modified'.

4 - A pasta 'sn9c103' contém uma versão consolidada em um único arquivo, porém somente com o código necessário para a câmera Elgin SN9C103 funcionar.

5 - A pasta 'udd' contém um template para criação de módulos de driver para dispositivos USB.
